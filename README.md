# In the project directory, you can run:
For starting the client:<br/>
cd frontend<br/>
cd client<br/>
run 'npm install'<br/>
run 'npm start' - it will start client automatically<br/>

For starting the server:<br/>
cd backend<br/>
run 'npm install'<br/>
run 'nodemon app.js'<br/>
